---
description: "Обновить прогресс по плану"
---

Обнови статус задач в плане: $ARGUMENTS

Формат аргументов: "имя-плана: задача №X → статус"
Примеры:
  /wiki-plan api-refactor: задача №3 → готово
  /wiki-plan sprint-12: задача №1 → в работе, задача №2 → заблокировано (ждём ревью)

Шаги:
1. Открой wiki/plans/{имя-плана}.md
2. Обнови статус указанных задач в таблице
3. Добавь запись в wiki/log.md
4. Покажи итоговую таблицу плана

Статусы: 🔲 план / 🔄 в работе / ✅ готово / ⏸ заблокировано

Если $ARGUMENTS не указан — покажи список всех планов из wiki/index.md.
