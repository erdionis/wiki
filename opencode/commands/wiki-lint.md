Выполни двухуровневый аудит вики проекта.

Сначала проверь `wiki/hot-context.md` — если он устарел (> 24ч), поставь пометку
и предложи обновить после аудита.

══════════════════════════════════════════
УРОВЕНЬ 1 — Авто-исправления (делать сразу)
══════════════════════════════════════════

- Файл в wiki/ отсутствует в index.md → добавить запись
- Запись в index.md на несуществующий файл → пометить [ОТСУТСТВУЕТ]
- Битая [[ссылка]] с единственным совпадением → исправить путь
- Очевидно связанные страницы без взаимных ссылок → добавить
- Страница со статусом **draft** без ошибок → продвинуть до **active**
- Страница со статусом **stale**, чьи источники не менялись → вернуть **active**

══════════════════════════════════════════
УРОВЕНЬ 2 — Доклад (не трогать, три яруса)
══════════════════════════════════════════

🔴 КРИТИЧНО — требует решения команды:
- Открытое противоречие в ## Противоречия старше 30 дней
- Страница [устарело] или статус **stale** старше 30 дней без обновления
- Метка из sources.md без ни одной wiki-страницы

🟡 ВНИМАНИЕ — исправить в ближайший спринт:
- Страница [изменено:] или [требует проверки]
- Страница со статусом **draft** старше 7 дней
- Осиротевшая страница (нет входящих ссылок)
- Компонент упоминается на страницах, но нет своей страницы modules/
- overview.md не обновлялась более 30 дней
- Страница с **Уверенность**: `inferred` и статусом `draft` → требует проверки человеком
- **Hub-stub**: страница с >5 входящими [[wikilinks]] и <300 символов контента
  → «страница X — узловая, но почти пустая; требует расширения»
- **Orphan cluster**: группа страниц, ссылающихся только друг на друга, без связи с основным графом
  → «изолированный кластер: страницы A, B, C — нет входящих ссылок из основного графа»

Как построить граф для hub-stubs и orphan clusters:
1. Пройти все `.md` в `wiki/`, собрать все `[[ссылки]]` → построить граф входящих рёбер
2. Hub-stub = узел с indegree > 5 И len(content) < 300 символов
3. Orphan cluster = связный компонент без рёбер в/из остальных страниц (кроме index.md)

🔵 ИНФОРМАЦИЯ — улучшить по возможности:
- Страница без блока ## Связанные страницы
- Страница без поля Источники
- Страница без поля **Статус** (не проставлен lifecycle)
- Страница без поля **Уверенность** (не проставлен confidence tag)
- Архивные страницы, чьи источники обновились
